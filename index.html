O que é front End?

Desenvolvedor Front End é responsável pelos códigos que o navegador processa do lado do usuário, elementos de um website como menu, telas, formulários, etc. serão feitos por desenvolvedor front end.

Lógica de Programação

A lógica de programação é o início da aprendizagem do desenvolvedor, juntamente dos algoritmos. Ao entender você terá muito mais facilidade para aprender novas linguagens, afinal, basicamente todas as linguagens modernas seguem o mesmo princípio de estrutura de dados.

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style

Javascript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web
